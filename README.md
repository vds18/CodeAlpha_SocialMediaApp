CodeAlpha Task 2: Mini Social Media App

This project is developed as part of the **CodeAlpha Full Stack Development Internship (M1)**.

The goal of this task is to build a **mini social media platform** with core features like user profiles, posts, comments, and a follow system.



📌 Project Objective

To design and develop a basic social media application that allows users to:

- Create and manage profiles  
- Share posts  
- Interact through likes and comments  
- Follow and connect with other users  



🚀 Features Implemented

- 👤 User Profiles  
- 📝 Create & View Posts  
- ❤️ Like Posts  
- 💬 Comment System  
- 👥 Follow Users  
- 📰 Personalized Feed (based on following)  
- 🔍 Discover Users  


🛠️ Tech Stack

**Frontend:**
- HTML  
- CSS  
- JavaScript  

**Backend:**
- Node.js  
- Express.js  

**Database:**
- MongoDB  

📂 Database Collections

- Users  
- Posts  
- Comments  
- Followers / Following
